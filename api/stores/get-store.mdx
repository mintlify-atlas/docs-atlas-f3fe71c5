---
title: Get store
description: Retrieve detailed information about a specific store by ID
---

## Endpoint

```
GET /stores/:id
```

Retrieve detailed information about a specific store, including owner contact information.

## Authentication

No authentication required. This is a public endpoint.

## Path parameters

<ParamField path="id" type="string" required>
  The unique identifier of the store (MongoDB ObjectId)
</ParamField>

## Response

<ResponseField name="success" type="boolean" required>
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Error message (only present when success is false)
</ResponseField>

<ResponseField name="data" type="object">
  Contains the response data (only present when success is true)
  
  <Expandable title="data properties">
    <ResponseField name="store" type="object" required>
      Store object with detailed information
      
      <Expandable title="store properties">
        <ResponseField name="id" type="string" required>
          Unique identifier for the store
        </ResponseField>
        
        <ResponseField name="_id" type="string" required>
          MongoDB ObjectId (same as id)
        </ResponseField>
        
        <ResponseField name="name" type="string" required>
          Store name
        </ResponseField>
        
        <ResponseField name="description" type="string">
          Store description
        </ResponseField>
        
        <ResponseField name="upi_id" type="string" required>
          UPI ID for payments
        </ResponseField>
        
        <ResponseField name="owner_id" type="string" required>
          ID of the store owner
        </ResponseField>
        
        <ResponseField name="ownerId" type="string" required>
          Store owner ID (camelCase)
        </ResponseField>
        
        <ResponseField name="is_active" type="boolean" required>
          Whether the store is active
        </ResponseField>
        
        <ResponseField name="isActive" type="boolean" required>
          Active status (camelCase)
        </ResponseField>
        
        <ResponseField name="operating_hours" type="object">
          Store operating hours configuration
        </ResponseField>
        
        <ResponseField name="operatingHours" type="object">
          Operating hours (camelCase)
        </ResponseField>
        
        <ResponseField name="image_url" type="string">
          URL to store image
        </ResponseField>
        
        <ResponseField name="imageUrl" type="string">
          Store image URL (camelCase)
        </ResponseField>
        
        <ResponseField name="qr_code_url" type="string">
          URL to store QR code
        </ResponseField>
        
        <ResponseField name="qrCodeUrl" type="string">
          QR code URL (camelCase)
        </ResponseField>
        
        <ResponseField name="owner_name" type="string">
          Name of the store owner
        </ResponseField>
        
        <ResponseField name="owner_email" type="string">
          Email of the store owner
        </ResponseField>
        
        <ResponseField name="owner_phone" type="string">
          Phone number of the store owner
        </ResponseField>
        
        <ResponseField name="created_at" type="string">
          ISO 8601 timestamp of creation
        </ResponseField>
        
        <ResponseField name="createdAt" type="string">
          Creation timestamp (camelCase)
        </ResponseField>
        
        <ResponseField name="updated_at" type="string">
          ISO 8601 timestamp of last update
        </ResponseField>
        
        <ResponseField name="updatedAt" type="string">
          Last update timestamp (camelCase)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

## Example request

<CodeGroup>
```bash cURL
curl https://api.campusbite.com/stores/507f1f77bcf86cd799439011
```

```javascript JavaScript
const storeId = '507f1f77bcf86cd799439011';
const response = await fetch(`https://api.campusbite.com/stores/${storeId}`);
const data = await response.json();
```

```python Python
import requests

store_id = '507f1f77bcf86cd799439011'
response = requests.get(f'https://api.campusbite.com/stores/{store_id}')
data = response.json()
```
</CodeGroup>

<Note>
Unlike the list stores endpoint, this endpoint returns store information regardless of the `is_active` status. The response includes additional owner contact details (email and phone number).
</Note>
