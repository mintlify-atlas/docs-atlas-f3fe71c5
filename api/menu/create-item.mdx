---
title: Create menu item
description: Add a new menu item to your store with optional image upload
---

## Endpoint

```
POST /api/menu
```

## Authentication

This endpoint requires authentication and store employee authorization.

<Note>
  Only store owners can create menu items. The system automatically associates the menu item with the authenticated user's store.
</Note>

## Request

This endpoint accepts `multipart/form-data` for image uploads.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Must be `multipart/form-data`
</ParamField>

### Body parameters

<ParamField body="name" type="string" required>
  Name of the menu item. Will be trimmed of whitespace.
</ParamField>

<ParamField body="price" type="number" required>
  Price of the menu item. Must be a positive number (minimum 0).
</ParamField>

<ParamField body="description" type="string">
  Optional description of the menu item.
</ParamField>

<ParamField body="category" type="string">
  Optional category for the menu item (e.g., "Burgers", "Drinks", "Desserts").
</ParamField>

<ParamField body="image" type="file">
  Optional image file for the menu item. Accepted formats depend on server configuration.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  Contains the created menu item data
  
  <ResponseField name="data.menuItem" type="object">
    The created menu item object
    
    <ResponseField name="data.menuItem.id" type="string">
      Unique identifier for the menu item
    </ResponseField>
    
    <ResponseField name="data.menuItem.store_id" type="string">
      ID of the store this item belongs to
    </ResponseField>
    
    <ResponseField name="data.menuItem.name" type="string">
      Name of the menu item
    </ResponseField>
    
    <ResponseField name="data.menuItem.description" type="string">
      Description of the menu item (null if not provided)
    </ResponseField>
    
    <ResponseField name="data.menuItem.price" type="number">
      Price of the menu item
    </ResponseField>
    
    <ResponseField name="data.menuItem.image_url" type="string">
      URL to the menu item image (null if not uploaded)
    </ResponseField>
    
    <ResponseField name="data.menuItem.category" type="string">
      Category of the menu item (null if not provided)
    </ResponseField>
    
    <ResponseField name="data.menuItem.is_available" type="boolean">
      Availability status (defaults to true)
    </ResponseField>
    
    <ResponseField name="data.menuItem.created_at" type="string">
      ISO 8601 timestamp of creation
    </ResponseField>
    
    <ResponseField name="data.menuItem.updated_at" type="string">
      ISO 8601 timestamp of last update
    </ResponseField>
  </ResponseField>
</ResponseField>

### Error responses

<Warning>
  If you don't own a store, you'll receive a 404 error.
</Warning>

#### 404 - No store found

```json
{
  "success": false,
  "message": "You do not own a store."
}
```

#### 401 - Unauthorized

```json
{
  "success": false,
  "message": "Authentication required."
}
```

#### 403 - Forbidden

```json
{
  "success": false,
  "message": "Insufficient permissions."
}
```
